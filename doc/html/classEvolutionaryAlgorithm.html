Detailed Description

Main library class controlling evolutionary algorithm process.

Stores collection of neural networks and using EvolutionaryAlgorithmParameters decides whether it should crossover or mutate individuals. User can determine shape of neural networks, crossover and mutation probabilities, weight variance, population size and how many children should be bred per generation.

Authors
Piotr Truszczynski, Jakub Fajkowski

PNeuralNetwork EvolutionaryAlgorithm::getNext ( )

The result of this method depends on current_network_ counter. If current network number is equal to population size - there are some offsprings added to population vector. After that algorithm will return children networks. If the last child is selected, counter resets and the weakest networks are killed.

Current neural network chosen by algorithm.
